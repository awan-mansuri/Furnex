# Furnex — Full-Stack Furniture E-Commerce Platform

Furnex is a Django-based e-commerce platform for a furniture store, 
covering the full shopping flow from browsing to payment to delivery tracking.

## Features
- Product catalog with categories, discounts, and stock tracking
- 3D/AR product preview via uploaded .glb model files
- Cart, checkout, and Razorpay payment gateway integration
- Order lifecycle tracking (placed → dispatched → delivered)
- PDF invoice generation
- Coupon codes, wishlist, and product comparison
- Product reviews and ratings
- Custom admin dashboard (Jazzmin) with sales analytics and order management
- Back-in-stock alerts and notifications

## Tech Stack
- Python, Django
- SQLite (dev) 
- Razorpay API
- Jazzmin (Django admin theme)
